\begin{center}
    \Large\textbf{关于本书}
\end{center}

\vspace{1cm}

本书为人工智能领域的学生、自学者和从业者提供了一本结构清晰、内容全面的神经网络入门教材。本书从数学基础和激活函数入手，逐步展开架构、优化、训练、数据处理和现代大规模模型的核心思想，特别注重概念清晰度和实际实现。

\textbf{主要特色：}
\begin{itemize}
    \item 从数学基础到 Transformer 和现代训练技术的完整13章学习路径
    \item 全书包含严谨的数学推导、直观的解释和精心设计的图表
    \item 以 PyTorch 生态系统为中心的实际实现示例
    \item 涵盖 CNN、RNN、跳跃连接和注意力等核心架构和机制
    \item 包含优化、训练流程、数据集处理和计算效率的专门章节
    \item 精心策划的参考文献库，连接经典论文与现代发展
\end{itemize}
